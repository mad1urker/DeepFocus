# DeepFocus

`DeepFocus` (`df`) — консольный трекер фокус-сессий.

## Что умеет

- запускать фокус-сессию;
- завершать активную сессию;
- показывать статус активной сессии;
- выводить логи всех сессий или только за сегодня;
- хранить состояние между вызовами через JSON-файл `deepfocus_log.json` рядом с исполняемым файлом.

## Команды

```bash
df start
df end
df status
df logs
df logs --today
df --help
```

## Формат хранения JSON

```json
[
  {
    "date": "2026-03-18",
    "start": "09:15:00",
    "end": "12:30:00"
  },
  {
    "date": "2026-03-18",
    "start": "13:10:00",
    "end": null
  }
]
```

## Сборка и запуск

```bash
dotnet build
dotnet run -- start
```

`deepfocus_log.json` создаётся автоматически при первой записи.
